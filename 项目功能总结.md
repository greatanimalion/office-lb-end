# 项目功能总结

> 基于 Express + Prisma(SQLite) + OnlyOffice 的局域网文档协同管理系统

## 已实现功能模块

### 1. 用户认证与账户管理
- 邮箱+密码注册/登录（密码 bcrypt 加密）
- 邮箱验证码发送（Redis + Nodemailer，5 分钟有效期，60 秒重试冷却）
- JWT 令牌鉴权（`Authorization` 请求头，无 Bearer 前缀）
- 用户列表查询、用户详情查询
- 管理员硬编码保障（邮箱 `15294745236@163.com` 启动时自动设为 admin）

### 2. 第三方 OAuth 登录
- GitLab OAuth2 登录（Passport 策略）
- 自动创建/更新临时账户（根据 provider + providerId）
- 支持钉钉、微信 OAuth 配置（仅配置项，未实现完整路由）
- GitLab 登录成功后携带 token 重定向到前端

### 3. 文档 CRUD
- 文档创建（上传文件 + 创建数据库记录）
- 文档更新（标题修改）
- 文档删除（级联删除版本文件 + 数据库记录）
- 文档列表查询（分页、按 owner_type 和 owner_id 过滤、标题模糊搜索）
- 文档详情获取
- 共享文档列表获取
- 文档流式下载（支持 docx/xlsx/pptx/pdf，Content-Disposition 中文编码）

### 4. 文档版本管理
- 文档版本创建（上传/编辑后自动创建版本节点）
- 版本号自增（从 1 开始，每次编辑 +1）
- 版本历史查询（含修改人用户名）
- 版本回滚（切换 documentVId 指向目标版本）
- 版本删除（删除指定版本记录）
- Document-Version 节点关联（`Document.documentVId` 指向当前版本）

### 5. 分片上传（大文件支持）
- 初始化上传会话（生成 fileId，创建临时目录）
- 逐片上传（支持 multipart、base64、raw buffer）
- 分片完整性校验（MD5 哈希比对）
- 分片合并（按索引顺序合并 + 全文件 MD5 校验）
- 上传进度查询（已上传分片 / 总分片）
- 断点续传（查询已上传分片 + 缺失分片列表）
- 取消上传（清理临时文件 + 删除会话）
- 上传会话列表
- 过期会话自动清理（24 小时）

### 6. 组管理
- 组的创建（自动将创建者设为 owner）
- 组信息更新（组名、描述）
- 组删除（级联删除成员关系）
- 用户所属组列表查询
- 组成员管理（添加成员、移除成员、成员列表）
- 用户切换所在组（`user.groupId` 更新）

### 7. 文件夹管理
- 文件夹创建（指定所属组、父文件夹、权限掩码）
- 文件夹树形结构查询（递归构建多级树）
- 文件夹列表查询（按组/父文件夹过滤）
- 文件夹更新（名称、权限）
- 文件夹删除（递归删除子文件夹）

### 8. 权限系统（位掩码方案）
- 9 种权限定义：VIEW / DOWNLOAD / EDIT / DELETE / COMMENT / CHANGE_PERMISSION / SHARE / MAKE_TEMPLATE / UPLOAD_FILE
- 权限位掩码存储（二进制位运算，如 `0b000000001` 表示查看）
- `permissonToNum()` / `numToPermisson()` 双向转换工具
- Permission 表支持：操作人、被授权人、目标文档、开始/结束时间窗口、密码校验、访问次数自减
- 权限校验链路：
  1. `public` 文档 → 所有人放行
  2. `admin` 角色 → 放行
  3. 文档所有者 → 放行
  4. Permission 记录校验（时间窗口 + 次数 + 密码 + 权限位）
  5. 组归属校验（ownerType === 'group' → 查 GroupMember）
  6. 文件夹归属校验（ownerType === 'folder' → 上溯找 group → 查 GroupMember）
- 文档/文件夹自带 `permission` 字段作为 fallback

### 9. OnlyOffice 在线编辑集成
- 编辑器配置生成（`GET /api/onlyoffice/:documentId/config`）
  - 文档类型/标题/URL/key
  - 用户信息（id, name, avatar）
  - 工作模式（edit/view）基于权限校验结果
  - 权限控制（edit/download/print/share/comment/review）
  - JWT Token 签名（使用 ONLYOFFICE_JWT_SECRET）
- 回调处理（`POST /api/onlyoffice/:documentId/callback`）
  - status=1: 正在编辑
  - status=2: 强制保存 → 从 OnlyOffice 下载文件 → 创建新版本（版本号+1）→ 更新 documentVId
  - status=3: 已保存
  - status=4: 关闭无修改
- 文档下载 URL 与 callback URL 基于 `NODE_SERVER_URL` 动态生成

### 10. 全文搜索
- MeiliSearch 集成（支持弹性降级到数据库模糊搜索）
- 可搜索属性：title / filename / content / tags
- 过滤条件：owner_id, 时间范围
- 排序：created_at, updated_at
- 文档内容文本提取（docx XML 解析）
- 标签建议（从标题中提取 `[标签]` 格式内容）
- 全文搜索初始化 + 索引更新/删除

### 11. 审计日志
- 审计记录自动写入（文档创建/修改/删除/分享/版本操作等）
- 审计日志查询（按 userId、documentId、action、时间范围过滤）
- 审计日志中间件（自动拦截成功响应并记录日志）
- 预定义审计动作 16 种 + 实体类型 5 种

### 12. 分享链接（外共享）
- 创建分享链接（UUID token，支持密码保护、过期时间、最大访问次数）
- 链接密码校验（PBKDF2 加密存储）
- 链接有效期/次数校验（过期或次数用完自动失效）
- 访问次数自增
- 链接删除
- 文档分享链接列表查询

### 13. 水印功能
- 文本水印添加（支持 docx/docx/xml/html/svg/txt/通用格式）
- docx 水印（通过 adm-zip 注入 word/document.xml 注释）
- 通用文件水印追加（注释/块注释形式）
- 水印信息提取（从文件内容正则匹配水印标记）
- 水印验证（比对用户 ID）
- 水印移除
- 访问 Token 生成/验证（Base64 编码，含过期时间）

### 14. OCR 文字识别
- 图片 OCR（Tesseract.js，支持英文 + 简体中文）
- PDF 文本提取（正则解析 PDF 裸文本）
- docx 文本提取（正则匹配 XML 标签）
- 多文件类型入口分发

### 15. 安全与限流
- API 全局限流（15 分钟 1000 次）
- 认证接口限流（15 分钟 1000 次）
- 验证码接口限流（1 分钟 1000 次）
- 上传接口限流（1 小时 20 次）
- 上传文件类型白名单（doc/docx/xls/xlsx/ppt/pptx/pdf/txt/rtf/odt/ods/odp）
- 上传文件大小限制（50MB）

### 16. 其他基础设施
- 配置化系统（`src/config/` 集中管理所有配置，基于 `.env` 环境变量）
- Express Session（24h + httpOnly）
- CORS 跨域配置
- 健康检查端点（`GET /health`）
- 优雅关闭（SIGTERM/SIGINT，10 秒超时强制退出）
- Winston 日志系统（控制台 + 文件，按级别分离 error.log / combined.log）
- 文件存储目录自动创建（documents/temp/versions/previews）
- 数据库 SQLite + Prisma ORM + libSQL adapter
- Redis 缓存（验证码存储）
- 错误处理中间件（统一错误响应，开发环境返回 stack trace）
- 404 未找到处理
- 图形水印（sharp 图片合成水印）
